# Tag Taxonomy

This file stores tags that have actually appeared in saved idea cards.

The taxonomy should grow slowly. Prefer reusing existing tags before creating new ones.

---

## How To Use This File

- Keep tags short and reusable.
- Use 3-4 tags per idea card.
- Add a new tag only when no existing tag fits.
- When a tag reaches 3 or more uses, move it into the core tag section during periodic cleanup.
- Counts do not need to be perfect after every capture; a monthly cleanup is usually enough.
