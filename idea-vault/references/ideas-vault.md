# Idea Vault

This file stores saved idea cards.

Keep cards in chronological order. The first saved card should be numbered `#001`, then `#002`, and so on.

---

## How To Use This File

When a new card is confirmed, paste it at the end of this file below the last card.

If this file has no saved cards yet, the next card is `#001` and the collision section should be `无`.
